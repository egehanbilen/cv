# Curriculum Vitae — Dr. Egehan Bilen

Single-file HTML curriculum vitae for **Dr. Egehan Bilen**, Specialist Physician in
Obstetrics & Gynecology (İzmir, Türkiye) and developer of
[ObGynAssist](https://obgynassist.com).

## Usage

- Open **`index.html`** in any modern browser.
- To export a PDF: open it and use **Print → Save as PDF**. A dedicated print
  stylesheet produces a clean, ink-friendly, properly paginated document.

## Notes

- Fully self-contained — the only external dependency is Google Fonts.
- Responsive (mobile → desktop), accessible (semantic landmarks, AA contrast,
  `prefers-reduced-motion`), and print-optimised.
